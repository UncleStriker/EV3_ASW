Descripción del proyecto: En esta 3ra evaluaión, actualizamos la pagina de la biblioteca "Tercer Ojo" para dar una cordial bienvenida a todos los lectores por la inaugración de la pagina.
  Dentro de esta se puede ver los 3 primeros libros de cada categoria, ciencia ficción y fantasía. Ademas, cada sección cuenta con un boton el cual carga la base de datos de todos los libros registrados en ella, pudiendo añadir nuevos libros, eliminar algun registro anterior y visualizar todos los registros al final de la OPA (one page aplication).
